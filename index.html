Reddit.reduce()

A stripped down, read-only version of Reddit.

Made using:

React / Redux / JavaScript

As part of Codecademy's full-stack ciriculum, a prompt was given to create a web app that interacts with Reddit's API in a read-only format. As a fan of alleteration and bad puns, I decided to create Reeddit.reduce() - a clean, simplified version of Reddit. Reddit.reduce() is built with React, Redux, and Tailwind and deployed on Netlify.
